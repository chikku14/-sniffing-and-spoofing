# Packet Sniffing and Spoofing

This project demonstrates basic packet sniffing and packet spoofing in a controlled lab environment using Python and Scapy.

## Files
- sniff.py: Captures packets
- spoof.py: Sends spoofed packets

## Run
```bash
python sniff.py
python spoof.py
```
 it is for learning and authorized environments only. That matches how these projects are usually presented in security labs
